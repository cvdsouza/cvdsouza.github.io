Gateway Oriented Reconfigurable Ecosystems

  • Learn More

A policy management module for an edge based computing framework. This project was presented as my thesis for the research conducted which was partially funded by Cisco Inc.

This research expands on the notion of bringing computational power to the edge-of-the-network, and then integrating it with the cloud computing paradigm whilst providing services to diverse IoT-based applications. This expansion is achieved through the establishment of a new computing model that serves as a platform for IoT-based devices to communicate with services in real-time. We name this paradigm as Gateway-Oriented Recon gurable Ecosystem (GORE) computing.

Collaborative Research Into Threats

- A project by MITRE

CRITs is an open source malware and threat repository that leverages other open source software to create a unified tool for analysts and security experts engaged in threat defense. It has been in development since 2010 with one goal in mind: give the security community a flexible and open platform for analyzing and collaborating on threat data.

One of the primary components that CRITS support is the notion of Services. These services extend the functionality of CRITS to run opensourced and vendor driven services via API's or customized installations. This allow analysts to further enrich their analysis of data ingested into CRITS. Custom services can be found here : CRITs services

Modern Honeypot Networks

From the secure deployment to the aggregation of thousands of events MHN provides enteprise grade management of the most current open source honeypot software. MHN is completely free open source software which supports external and internal honeypot deployments at a large and distributed scale. MHN uses the HPFeeds standard and low-interaction honeypots to keep effectiveness and security at enterprise grade levels.
